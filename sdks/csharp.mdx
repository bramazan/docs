# C# / .NET SDK

<Warning>
The C# SDK is currently in development. Use the REST API directly in the meantime.
</Warning>

## Expected Usage

### Installation

```bash
dotnet add package Odeal.PaymentSdk
```

### Basic Usage

```csharp
using Odeal;

var odeal = new OdealClient(new OdealOptions
{
    ApiKey = Environment.GetEnvironmentVariable("ODEAL_API_KEY"),
    Environment = OdealEnvironment.Production
});

// Create payment
var payment = await odeal.Payments.CreateAsync(new PaymentCreateRequest
{
    MerchantId = "merchant-123",
    Amount = 100.00m,
    Currency = "TRY",
    OrderId = "order-456"
});

// Process with card
var result = await odeal.Payments.ProcessAsync(payment.Id, new PaymentProcessRequest
{
    Card = new CardDetails
    {
        Number = "4111111111111111",
        HolderName = "John Doe",
        ExpiryMonth = "12",
        ExpiryYear = "2025",
        Cvv = "123"
    }
});

Console.WriteLine(result.Status); // "SUCCEEDED"
```

## Current Alternative

```csharp
using System.Net.Http;
using System.Text;
using System.Text.Json;

var client = new HttpClient();
client.DefaultRequestHeaders.Authorization = 
    new AuthenticationHeaderValue("Bearer", Environment.GetEnvironmentVariable("ODEAL_API_KEY"));

var request = new
{
    merchantId = "merchant-123",
    amount = 100.00,
    currency = "TRY",
    orderId = "order-456"
};

var content = new StringContent(
    JsonSerializer.Serialize(request),
    Encoding.UTF8,
    "application/json"
);

var response = await client.PostAsync(
    "https://api.odeal.com/v1/payments",
    content
);

var payment = await response.Content.ReadFromJsonAsync<Payment>();
```
