## Expected Usage

### Installation

```bash
pip install odeal-python
```

### Basic Usage

```python
from odeal import Odeal

# Initialize client
odeal = Odeal(api_key=os.environ['ODEAL_API_KEY'])

# Create payment
payment = odeal.payments.create(
    merchant_id='merchant-123',
    amount=100.00,
    currency='TRY',
    order_id='order-456'
)

# Process with card
result = odeal.payments.process(
    payment.id,
    card={
        'number': '4111111111111111',
        'holder_name': 'John Doe',
        'expiry_month': '12',
        'expiry_year': '2025',
        'cvv': '123'
    }
)

print(result.status) # 'SUCCEEDED'
```

### Async Usage

```python
import asyncio
from odeal import AsyncOdeal

async def create_payment():
    odeal = AsyncOdeal(api_key=os.environ['ODEAL_API_KEY'])
    
    payment = await odeal.payments.create(
        merchant_id='merchant-123',
        amount=100.00,
        currency='TRY'
    )
    
    return payment

asyncio.run(create_payment())
```

## Current Alternative: REST API

```python
import requests

def create_payment(data):
    response = requests.post(
        'https://api.odeal.com/v1/payments',
        headers={
            'Authorization': f'Bearer {os.environ["ODEAL_API_KEY"]}',
            'Content-Type': 'application/json'
        },
        json=data
    )
    
    return response.json()

payment = create_payment({
    'merchantId': 'merchant-123',
    'amount': 100.00,
    'currency': 'TRY',
    'orderId': 'order-456'
})
```
