# PHP SDK

<Warning>
The PHP SDK is currently in development. Use the REST API directly in the meantime.
</Warning>

## Expected Usage

### Installation

```bash
composer require odeal/payment-sdk
```

### Basic Usage

```php
<?php
use Odeal\OdealClient;

$odeal = new OdealClient([
    'api_key' => getenv('ODEAL_API_KEY'),
    'environment' => 'production'
]);

// Create payment
$payment = $odeal->payments->create([
    'merchantId' => 'merchant-123',
    'amount' => 100.00,
    'currency' => 'TRY',
    'orderId' => 'order-456'
]);

// Process
$result = $odeal->payments->process($payment->id, [
    'card' => [
        'number' => '4111111111111111',
        'holderName' => 'John Doe',
        'expiryMonth' => '12',
        'expiryYear' => '2025',
        'cvv' => '123'
    ]
]);

echo $result->status; // 'SUCCEEDED'
?>
```

## Current Alternative

```php
<?php
$ch = curl_init('https://api.odeal.com/v1/payments');

curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Authorization: Bearer ' . getenv('ODEAL_API_KEY'),
    'Content-Type: application/json'
]);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode([
    'merchantId' => 'merchant-123',
    'amount' => 100.00,
    'currency' => 'TRY'
]));
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$payment = json_decode($response);

curl_close($ch);
?>
```
