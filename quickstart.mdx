# Quickstart Guide

This guide will help you make your first payment using √ñdeal Payment Infrastructure.

## Prerequisites

Before you begin, ensure you have:
- API credentials (API key and merchant ID)
- Development environment set up
- Basic understanding of REST APIs

<Note>
Don't have API credentials? Contact our support team at support@odeal.com
</Note>

## Step 1: Set Up Your Environment

Store your API credentials securely:

<CodeGroup>
```bash Environment Variables
export ODEAL_API_KEY="test_sk_xxxxxxxxxxxxxxxx"
export ODEAL_MERCHANT_ID="merchant_123456"
export ODEAL_API_URL="https://api.odeal.com/v1"
```

Let's create a simple non-3D payment:

<CodeGroup>
```bash cURL
curl -X POST https://api.odeal.com/v1/payments \
  -H "X-API-Key: test_sk_xxxxxxxxxxxxxxxx" \
  -H "X-Idempotency-Key: $(uuidgen)" \
  -H "Content-Type: application/json" \
  -d '{
    "merchantId": "merchant_123456",
    "amount": {
      "amount": 100.00,
      "currency": "TRY"

    "description": "Test payment"

```

```javascript Node.js
const axios = require('axios');
const { v4: uuidv4 } = require('uuid');
const config = require('./config');

async function createPayment() {
  try {
    const response = await axios.post(`${config.apiUrl}/payments`, {
      merchantId: config.merchantId,
      amount: {
        amount: 100.00,
        currency: 'TRY'
      },
      cardDetails: {
        cardNumber: '5555444433332222',
        cardHolderName: 'JOHN DOE',
        expiryMonth: '12',
        expiryYear: '2025',
        cvv: '123'
      },
      operationType: 'SALE',
      installments: 1,
      threeDSecure: false,
      description: 'Test payment'
    }, {
      headers: {
        'X-API-Key': config.apiKey,
        'X-Idempotency-Key': uuidv4(),
        'Content-Type': 'application/json'
      }
    });

    console.log('Payment successful:', response.data);
    return response.data;
  } catch (error) {
    console.error('Payment failed:', error.response?.data || error.message);
    throw error;
  }
}

createPayment();
```

```python Python
import requests
import uuid
from config import API_KEY, MERCHANT_ID, API_URL

def create_payment():
    url = f"{API_URL}/payments"
    headers = {
        'X-API-Key': API_KEY,
        'X-Idempotency-Key': str(uuid.uuid4()),
        'Content-Type': 'application/json'
    }
    
    payload = {
        'merchantId': MERCHANT_ID,
        'amount': {
            'amount': 100.00,
            'currency': 'TRY'
        },
        'cardDetails': {
            'cardNumber': '5555444433332222',
            'cardHolderName': 'JOHN DOE',
            'expiryMonth': '12',
            'expiryYear': '2025',
            'cvv': '123'
        },
        'operationType': 'SALE',
        'installments': 1,
        'threeDSecure': False,
        'description': 'Test payment'

    response = requests.post(url, json=payload, headers=headers)
    response.raise_for_status()

if __name__ == '__main__':
```

```csharp C#
using System;
using System.Net.Http;
using System.Text;
using System.Text.Json;
using System.Threading.Tasks;

public class PaymentService
{
    private readonly HttpClient _httpClient;
    private readonly AppSettings _settings;

    public PaymentService(AppSettings settings)
    {
        _settings = settings;
        _httpClient = new HttpClient
        {
            BaseAddress = new Uri(_settings.ApiUrl)
        };
        _httpClient.DefaultRequestHeaders.Add("X-API-Key", _settings.ApiKey);
    }

    public async Task<PaymentResponse> CreatePayment()
    {
        var request = new PaymentRequest
        {
            MerchantId = _settings.MerchantId,
            Amount = new Amount { Amount = 100.00m, Currency = "TRY" },
            CardDetails = new CardDetails
            {
                CardNumber = "5555444433332222",
                CardHolderName = "JOHN DOE",
                ExpiryMonth = "12",
                ExpiryYear = "2025",
                Cvv = "123"
            },
            OperationType = "SALE",
            Installments = 1,
            ThreeDSecure = false,
            Description = "Test payment"
        };

        var content = new StringContent(
            JsonSerializer.Serialize(request),
            Encoding.UTF8,
            "application/json"
        );

        var httpRequest = new HttpRequestMessage(HttpMethod.Post, "/payments");
        httpRequest.Headers.Add("X-Idempotency-Key", Guid.NewGuid().ToString());
        httpRequest.Content = content;

        var response = await _httpClient.SendAsync(httpRequest);
        response.EnsureSuccessStatusCode();

        var result = await response.Content.ReadAsStringAsync();
        return JsonSerializer.Deserialize<PaymentResponse>(result);
    }
}
```

## Step 5: Set Up Webhooks (Recommended)

Configure webhooks to receive real-time payment notifications:

1. Create a webhook endpoint in your application
2. Register it with √ñdeal (contact support)
3. Handle webhook events

```javascript Example Webhook Handler
app.post('/webhooks/payment', (req, res) => {
  const event = req.body;
  
  // Verify webhook signature (implement this!)
  if (!verifySignature(req)) {
    return res.status(401).send('Invalid signature');
  }
  
  switch (event.type) {
    case 'payment.succeeded':
      handlePaymentSuccess(event.data);
      break;
    case 'payment.failed':
      handlePaymentFailure(event.data);
      break;
    case 'payment.refunded':
      handlePaymentRefund(event.data);
      break;
  }
  
  res.status(200).send('OK');
});
```
