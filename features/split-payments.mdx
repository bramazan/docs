# Split Payments

Distribute payment amounts between sellers and platforms automatically.

<Info>
For comprehensive marketplace documentation, see [Marketplace Features](/features/marketplace)
</Info>

## Overview

Split payments enable:
- Revenue sharing between platform and sellers
- Multi-party payment distribution
- Automated commission calculation
- Individual settlements

## Quick Example

```json
{
  "amount": 10000,
  "currency": "TRY",
  "split": [
    {
      "sub_merchant_id": "sub_seller",
      "amount": 8500,
      "type": "seller_revenue"
    },
    {
      "sub_merchant_id": "mrc_platform",
      "amount": 1500,
      "type": "platform_commission"
    }
  ]
}
```
