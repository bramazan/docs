# Pay by Link

Generate secure payment links that customers can use to complete payments without integrating checkout pages.

## Overview

Pay by Link allows you to:
- Create payment links via API
- Share links via SMS, email, or social media
- Track link usage and expiration
- Customize payment pages

<Note>
This feature is currently in beta. Contact sales@odeal.com for access.
</Note>

## Creating a Payment Link

```bash
POST /api/v1/payment-links
```

```json
{
  "amount": 25000,
  "currency": "TRY",
  "description": "Premium subscription - Annual",
  "customer_email": "customer@example.com",
  "expires_at": "2025-01-31T23:59:59Z",
  "max_uses": 1,
  "success_url": "https://yoursite.com/success",
  "cancel_url": "https://yoursite.com/cancel"
}
```

Response:
```json
{
  "id": "link_abc123",
  "url": "https://pay.odeal.com.tr/l/abc123xyz",
  "status": "active",
  "amount": 25000,
  "currency": "TRY",
  "created_at": "2025-01-15T10:30:00Z",
  "expires_at": "2025-01-31T23:59:59Z"
}
```
