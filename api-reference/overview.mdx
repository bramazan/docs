# API Genel Bakış

Ödeal Payment Infrastructure API'sine hoş geldiniz. RESTful API'miz ödeme kabul etmenizi, üye işyerleri yönetmenizi ve işlem verilerine erişmenizi sağlar.

## Temel URL

<CodeGroup>
```bash Production
https://api.odeal.com.tr/v1
```

```bash Sandbox (Test)
https://sandbox-api.odeal.com.tr/v1
```
</CodeGroup>

## Kimlik Doğrulama

Tüm API istekleri API anahtarı ve HMAC imzası ile kimlik doğrulama gerektirir.

Detaylar için [Kimlik Doğrulama](/api-reference/authentication) sayfasına bakın.

## İstek Formatı

```http
POST /v1/payments HTTP/1.1
Host: api.odeal.com.tr
Content-Type: application/json
Authorization: Bearer API_ANAHTARINIZ
X-Idempotency-Key: benzersiz-istek-id
X-API-Version: 1.0

{
  "amount": 10000,
  "currency": "TRY"
}
```

## Yanıt Formatı

```json
{
  "id": "pay_abc123",
  "status": "completed",
  "amount": 10000,
  "currency": "TRY",
  "createdAt": "2025-12-19T18:30:00Z"
}
```

## API Yetenekleri

- **3D Secure**: Tam 3DS 1.0 ve 2.0 desteği
- **Çoklu Para Birimi**: TRY, USD, EUR kabul
- **Marketplace**: Split payment ve alt üye işyerleri
- **Idempotency**: Duplike işlemleri önleme
- **Webhook'lar**: Gerçek zamanlı event bildirimleri
- **Versiyonlama**: URL tabanlı API versiyonlama

## Rate Limitler

| Plan | İstek/Dakika | Burst |
