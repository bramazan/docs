## Hoş Geldiniz

Payven API, RESTful mimarisi üzerine kurulmuştur ve JSON formatında veri alışverişi yapar. Tüm istekler standart HTTP metodlarını (GET, POST, PUT, DELETE vb.) kullanır.

## Kimlik Doğrulama

API'ye erişim için Bearer Token tabanlı kimlik doğrulama kullanılır. Token almak için `/api/auth/login` uç noktasını kullanabilirsiniz.

"Authorization": "Bearer <token>"
