# Taksit API

Kart BIN numarasına ve tutara göre uygun taksit seçeneklerini sorgulama.

## Taksit Seçeneklerini Getir

### `GET /v1/installments`

Belirli bir kart ve tutar için kullanılabilir taksit seçeneklerini getirir.

```bash
GET /v1/installments?binNumber=540061&price=1000&currency=TRY
Authorization: Bearer API_ANAHTARINIZ
```

#### Query Parametreleri

| Parametre | Tip | Zorunlu | Açıklama |
|-----------|-----|---------|----------|
| `binNumber` | string | Evet | Kart BIN numarası (6-8 hane) |
| `price` | decimal | Evet | İşlem tutarı |
| `currency` | string | Hayır | Para birimi (varsayılan: TRY) |

## Kullanım Senaryosu

### Ödeme Formunda Taksit Gösterimi

```javascript
async function loadInstallmentOptions(cardNumber, amount) {
  const bin = cardNumber.substring(0, 6);
  
  const response = await fetch(
    `https://api.odeal.com.tr/v1/installments?binNumber=${bin}&price=${amount}&currency=TRY`,
    {
      headers: {
        'Authorization': `Bearer ${API_KEY}`
      }
    }
  );
  
  const data = await response.json();
  return data.installmentOptions;
}

// HTML gösterimi
function renderInstallmentOptions(options) {
  return options.map(option => `
    <option value="${option.installmentCount}">
      ${option.installmentCount}x ${option.installmentAmount.toFixed(2)} TL
      ${option.commissionAmount > 0 ? `(+${option.commissionAmount.toFixed(2)} TL)` : ''}
    </option>
  `).join('');
}
```

## Taksit Kuralları

### Minimum Tutarlar

| Taksit Sayısı | Minimum Tutar (TRY) |
|---------------|---------------------|
| 2-3 | 100 |
| 4-6 | 500 |
| 7-9 | 1,000 |
| 10-12 | 2,000 |

<Note>
Minimum tutarlar banka ve kart tipine göre değişiklik gösterebilir.
</Note>

### Komisyon Oranları

Taksit komisyon oranları:
- **Tek çekim**: %0 komisyon
- **2-3 taksit**: %1-2 komisyon
- **4-6 taksit**: %3-5 komisyon
- **7-9 taksit**: %6-8 komisyon
- **10-12 taksit**: %9-12 komisyon

<Warning>
Komisyon oranları banka anlaşmalarına göre değişkenlik gösterir. Gerçek oranlar için API yanıtını kullanın.
</Warning>

## Örnekler

### React Component

```jsx
import React, { useState, useEffect } from 'react';

function InstallmentSelector({ cardNumber, amount }) {
  const [options, setOptions] = useState([]);
  const [selected, setSelected] = useState(1);
  
  useEffect(() => {
    if (cardNumber.length >= 6) {
      loadInstallments();
    }
  }, [cardNumber, amount]);
  
  async function loadInstallments() {
    const bin = cardNumber.substring(0, 6);
    const response = await fetch(
      `/api/v1/installments?binNumber=${bin}&price=${amount}`
    );
    const data = await response.json();
    setOptions(data.installmentOptions);
  }
  
  return (
    <select value={selected} onChange={(e) => setSelected(e.target.value)}>
      {options.map(opt => (
        <option key={opt.installmentCount} value={opt.installmentCount}>
          {opt.installmentCount}x {opt.installmentAmount.toFixed(2)} TL
          {opt.totalAmount > amount && ` (+${(opt.totalAmount - amount).toFixed(2)} TL)`}
        </option>
      ))}
    </select>
  );
}
```

### Vue.js Component

```vue
<template>
  <select v-model="selectedInstallment">
    <option v-for="opt in installmentOptions" :key="opt.installmentCount" 
            :value="opt.installmentCount">
      {{ opt.installmentCount }}x {{ opt.installmentAmount.toFixed(2) }} TL
      <span v-if="opt.commissionAmount > 0">
        (+{{ opt.commissionAmount.toFixed(2) }} TL)
      </span>
    </option>
  </select>
</template>

<script>
export default {
  props: ['cardNumber', 'amount'],
  data() {
    return {
      installmentOptions: [],
      selectedInstallment: 1
    };
  },
  watch: {
    cardNumber() {
      if (this.cardNumber.length >= 6) {
        this.fetchInstallments();
      }
    }
  },
  methods: {
    async fetchInstallments() {
      const bin = this.cardNumber.substring(0, 6);
      const response = await fetch(
        `/api/v1/installments?binNumber=${bin}&price=${this.amount}`
      );
      const data = await response.json();
      this.installmentOptions = data.installmentOptions;
    }
  }
};
</script>
```

## Hata Kodları

| Kod | Açıklama | Çözüm |
|-----|----------|--------|
| `INVALID_BIN` | Geçersiz BIN numarası | 6-8 haneli BIN girin |
| `BIN_NOT_FOUND` | BIN bulunamadı | Kart bilgilerini kontrol edin |
| `AMOUNT_BELOW_MINIMUM` | Tutar minimum limitin altında | Tutarı artırın |
| `INSTALLMENTS_NOT_AVAILABLE` | Bu kart taksit desteklemiyor | Tek çekim kullanın |
