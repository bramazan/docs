## Reporting Issues

### Bug Reports

<Steps>
  <Step title="Check Known Issues">
    Review the [changelog](/resources/changelog) and [status page](/resources/status) for known issues
  </Step>
  
  <Step title="Gather Information">
    Collect relevant details:
    - Request ID / Transaction ID
    - Timestamp
    - Error message and code
    - Steps to reproduce
    - Expected vs actual behavior
  </Step>
  
  <Step title="Submit Report">
    Email [technical@odeal.com](mailto:technical@odeal.com) with:
    - Clear subject line
    - Detailed description
    - Code samples (if applicable)
    - Environment (test/production)
  </Step>
</Steps>

### Security Vulnerabilities

<Warning>
**DO NOT** report security issues publicly. Email [security@odeal.com](mailto:security@odeal.com) directly.
</Warning>

We take security seriously. Responsible disclosure process:

1. **Report**: Email security concerns to [security@odeal.com](mailto:security@odeal.com)
2. **Acknowledgment**: We'll respond within 48 hours
3. **Investigation**: We'll investigate and provide updates
4. **Resolution**: We'll fix the issue and credit you (if desired)
5. **Disclosure**: Public disclosure after fix is deployed

### Feature Requests

## SLA & Response Times

### Incident Priority Levels

| Priority | Description | Response Time | Resolution Target |
|----------|-------------|---------------|-------------------|
| **P0 - Critical** | Production down, payments failing | 15 minutes | 2 hours |
| **P1 - High** | Major feature degradation | 1 hour | 8 hours |
| **P2 - Medium** | Minor issue, workaround available | 4 hours | 24 hours |
| **P3 - Low** | General questions, feature requests | 24 hours | 5 business days |

### Escalation Process

If your issue isn't being resolved:

1. **Initial Contact**: Submit via normal support channels
2. **Follow-up**: If no response within SLA, reply to the ticket
3. **Escalation**: For P0/P1 issues, call phone support
4. **Management**: For urgent unresolved issues, email [escalations@odeal.com](mailto:escalations@odeal.com)
