## Incident History

### December 2025

<Accordion title="No incidents to report" icon="check-circle">
All systems operated normally throughout December 2025.
</Accordion>

### November 2025

<Accordion title="Webhook Delivery Delay - Nov 15, 2025" icon="exclamation-triangle">

**Status**: Resolved

**Duration**: 45 minutes (14:30 - 15:15 UTC+3)

**Impact**: Webhook deliveries were delayed by 5-10 minutes. No webhooks were lost.

**Root Cause**: RabbitMQ message broker experienced memory pressure due to a sudden spike in traffic.

**Resolution**: Scaled up RabbitMQ cluster and implemented better memory management.

**Prevention**: Added auto-scaling triggers and improved monitoring alerts.

</Accordion>

### October 2025

<Accordion title="Database Performance Degradation - Oct 8, 2025" icon="exclamation-triangle">

**Status**: Resolved

**Duration**: 2 hours (09:00 - 11:00 UTC+3)

**Impact**: API response times increased by 2-3x. No data loss.

**Root Cause**: Long-running analytics query caused table locks.

**Resolution**: Killed long-running query and optimized analytics queries to use read replicas.

**Prevention**: Implemented query timeout limits and separated analytics workload.

</Accordion>
